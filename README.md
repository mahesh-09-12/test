# 🚀 MERN Hackathon Project

Welcome to our hackathon project! This repository contains a **MERN stack** app built with:

- **MongoDB** (database)
- **Express.js** (backend)
- **React.js** (frontend)
- **Node.js** (runtime)

---

## 📦 Getting Started

### 1. Clone the repo

```
git clone https://github.com/mahesh-09-12/test.git
cd test

npm install
npm run dev
```
